# AI Chat Flutter

Многостраничное Flutter-приложение для работы с языковыми моделями через OpenRouter и VSEGPT.

## Возможности

- 💬 Общение с AI-моделями
- ⚙️ Выбор провайдера (OpenRouter / VSEGPT)
- 🔑 Настройка и хранение API-ключей
- 📊 Статистика использования токенов
- 📈 График расходов по дням
- 🗄️ Локальное хранение истории сообщений
- 🎨 Современный Material Design интерфейс

## Структура приложения

Приложение состоит из 4 основных экранов:

### 1. Главная страница
Основной интерфейс общения с языковыми моделями.

Функционал:
- отправка сообщений;
- просмотр истории диалога;
- выбор модели;
- отображение ответов модели.

### 2. Настройки провайдера
Экран настройки подключения к API.

Функционал:
- выбор провайдера (OpenRouter / VSEGPT);
- ввод API-ключей;
- сохранение настроек;
- настройка параметров подключения.

### 3. Статистика использования
Экран аналитики использования моделей.

Функционал:
- статистика токенов;
- информация по моделям;
- суммарное использование;
- история запросов.

### 4. График расходов
Визуализация затрат на использование моделей.

Функционал:
- отображение расходов по дням;
- динамика затрат;
- анализ использования API.

## Настройка

Перед запуском необходимо создать файл `.env` в корне проекта.

Пример содержимого:

```env
PROVIDER=OpenRouter
OPENROUTER_API_KEY=
VSEGPT_API_KEY=
BASE_URL=https://openrouter.ai/api/v1
MAX_TOKENS=1000
TEMPERATURE=0.7
```

Также рекомендуется создать файл `.env.example`:

```env
PROVIDER=OpenRouter
OPENROUTER_API_KEY=
VSEGPT_API_KEY=
BASE_URL=https://openrouter.ai/api/v1
MAX_TOKENS=1000
TEMPERATURE=0.7
```

После клонирования проекта:

```bash
cp .env.example .env
```

и заполнить необходимые значения.
